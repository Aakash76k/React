# ⏱️ Stopwatch using React useEffect

A Stopwatch built with **React**, **useState**, and **useEffect**.

## 🚀 Features

- Start Timer
- Stop Timer
- Reset Timer
- Displays Hours
- Displays Minutes
- Displays Seconds
- Displays Centiseconds
- Uses `setInterval()`
- Cleanup using `clearInterval()`

## 🛠️ Technologies Used

- React
- useState
- useEffect
- CSS

## 📚 Concepts Learned

- React Hooks
- State Management
- useEffect
- setInterval()
- clearInterval()
- Conditional Rendering
- Event Handling
